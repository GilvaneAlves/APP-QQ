📘 Ajuda – ScannerQQ

🧾 Sobre o Projeto

ScannerQQ é uma aplicação web que permite identificar produtos através da leitura de códigos de barras (EAN-13) ou digitando manualmente o código EAN ou o número do material. Ele consulta uma base local (produtos.json) e retorna a descrição do item com um link direto para o site da Quero‑Quero.

🔧 Funcionalidades

  • 📷 Leitura de códigos de barras usando a câmera
  • ⌨ Busca manual por código EAN
  • 🛠 Busca por número do material
  • 🔗 Link direto para o site da Quero‑Quero
  • ✅ Modo Celular e Desktop

📲 Como Usar

Modo Celular:

  1. Clique em Modo Celular
  2. Aponte a câmera para um código de barras
  3. Produto será identificado e exibido

Busca Manual:

  • Digite o código EAN ou o número do material
  • Clique em Buscar ou pressione Enter

💾 Estrutura da base (produtos.json)

[
  [
    {
      "Material": "123456",
      "Código EAN/UPC": "7891234567890",
      "Texto breve material": "Produto Exemplo"
    }
  ]
]

🧪 Tecnologias Usadas

  • JavaScript Puro (Vanilla)
  • ZXing (Scanner de Código de Barras)
  • HTML5, CSS3
  • API MediaDevices

⚠️ Problemas Comuns

Problema Causa Solução
❌ "Erro ao acessar a câmera" Permissão negada ou navegador Use o Chrome e permita o acesso
❌ Código EAN não encontrado Não está na base local Atualize o arquivo produtos.json
❌ Material não encontrado Material digitado incorretamente Verifique a digitação
